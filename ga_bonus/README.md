# Бонусный вариант генетического алгоритма

## Как использовать

```
python main.py
gcc fib.c
.\a.exe
```

## Пояснения

`fib.c` - результирующий алгоритм, вычисляющий числа Фибоначчи.

Файл использует расширения GCC, поэтому компилировать нужно именно этим компилятором.
Это связано с тем, что сгенерированный в `fib.c` алгоритм представляет собой одно выражение,
а не несколько, а значит, цикл `for` должен быть выражением — что не соответствует ANSI C, но допустимо в расширениях GCC.
